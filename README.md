# Oldagram

A small social media feed project built with vanilla HTML, CSS, and JavaScript.

This was mostly a learning project to practice building a UI that feels interactive, while keeping everything in plain frontend code (no frameworks).

---

# Live demo
https://philipvagne.github.io/Oldagram/

---

# What this is

Oldagram is basically an Instagram-style feed clone.  
Nothing fancy in terms of backend or data — everything is handled on the frontend using JavaScript objects.

The focus wasn’t on building something “big”, but more on making the UI feel alive and understanding how real apps manage simple interactions.

---

# Features

- Feed rendered dynamically from JavaScript
- Like button with live counter updates
- Double-click image to like (Instagram-style behavior)
- Stories section with “seen” state
- Dark mode toggle
- Bottom navigation with active state feedback
- Small UI animations and hover interactions

---

# Built with

- HTML
- CSS (Flexbox, variables, transitions)
- Vanilla JavaScript

---

# What I learned

This project helped me understand a few important things:

- how to structure a UI into separate parts (nav, feed, stories)
- how state works in a simple JavaScript app
- how to update the DOM without reloading everything
- how much difference small CSS interactions make to “feel”

---

# Things I’d improve next

If I come back to this, I’d probably:

- store likes and theme in localStorage so they persist
- break the JavaScript into modules instead of one file
- maybe connect it to a real API instead of static data
- improve accessibility (keyboard navigation, etc.)
